# Suggested Commands

## Development
```bash
npm run dev:server   # Start backend server with hot reload (tsx watch)
npm run dev:client   # Start frontend dev server (Vite)
```

## Testing
```bash
npm test            # Run all tests with Jest
npm run test:watch  # Run tests in watch mode
```

## Building
```bash
npm run build:server  # Build backend TypeScript
npm run build:client  # Build frontend with Vite
npm run build         # Build both server and client
```

## Code Quality
```bash
npm run lint    # Run ESLint on .ts and .tsx files
npm run format  # Format code with Prettier
```

## System Commands (macOS/Darwin)
- `ls -la` - List files with details
- `grep -r "pattern" .` - Search in files
- `find . -name "*.ts"` - Find TypeScript files
- `cat file.ts` - View file contents
- `rm -rf dir` - Remove directory
- `cp -r src dest` - Copy directory
